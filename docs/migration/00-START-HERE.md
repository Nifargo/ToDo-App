## ❓ Часті Питання

### Q: Скільки це займе часу?
**A:** ~50 годин загалом = 3-4 тижні при 2-3 год/день

### Q: Чи потрібно знати React?
**A:** Базові знання бажані. Є навчальні ресурси в 03-plan.md

### Q: Чи можна мігрувати поступово?
**A:** Ні, це повна міграція. Vanilla JS → React за один раз

### Q: Що з Firebase?
**A:** Firebase залишається без змін, тільки оновимо інтеграцію

### Q: Чи працюватиме PWA?
**A:** Так! Vite має vite-plugin-pwa з коробки

### Q: Що з bundle size?
**A:** Збільшиться з 53KB → ~140KB, але це OK для PWA
